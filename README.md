# Gamificação de Rotinas (Projeto de Software)

## 📌 Visão Geral
Sistema Full-Stack desenvolvido para a disciplina de Projeto de Software. O objetivo é aplicar mecânicas de gamificação (atribuição de XP, prazos e recompensas) na gestão de rotinas e tarefas domésticas, transformando obrigações em dados estruturados e analisáveis.

## ⚙️ Arquitetura e Stack Tecnológico
O projeto obedece à estrutura de 3 camadas:
- **Front-end:** Interface de utilizador em HTML5 e CSS3 (Dark Mode otimizado para legibilidade).
- **Back-end:** Motor de regras de negócio e persistência desenvolvido em **PHP 8+**, utilizando **PDO** e *Prepared Statements* para neutralizar vulnerabilidades de SQL Injection.
- **Base de Dados:** Modelo relacional normalizado em **MySQL**, garantindo a integridade transacional dos utilizadores, tarefas e histórico de XP.

## 📅 Roadmap de Entregas (Sprints)
- **[AC1] 15/03:** Fundação Arquitetural, esquema relacional da base de dados, interface de captura e motor de persistência (CRUD - Create).
- **[AC2] 12/04:** Implementação da listagem de dados, validação de status e motor de cálculo matemático de XP por cumprimento de prazos.
- **[AC3] 10/05:** Desenvolvimento do módulo de Resgate de Recompensas e dedução de saldo relacional.
- **[FINAL] 07/06:** Refatoração de código, tratamento avançado de sessões e implementação de Dashboard Analítico (BI) de produtividade.
